## Работа с process

В проекте для взаимодействия с окружением и аргументами командной строки используются возможности глобального объекта `process`:

- `process.argv` — чтение аргументов запуска (в коде: `argv[2]` — имя, `argv[3]` — величина в дюймах).
- `process.env` — чтение переменных окружения (в коде: `COLORIZE` для включения/выключения цветного вывода). Значения можно задавать через `.env` (dotenv).
- `process.exit(code)` — преждевременное завершение процесса с кодом ошибки при неверных входных данных.
- Для отладки выводится значение переменной окружения: `JSON.stringify(process.env.COLORIZE)`.

Также в коде выполняется валидация аргумента (`parseFloat`, `isNaN`) и расчёт (перевод дюймов в сантиметры).
